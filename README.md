# vue-tauri-test

This is an example project to show how to combine vue and tauri to build a native app.

## Requirements

- npm
- tauri (see [here](https://github.com/tauri-apps/tauri/wiki))

## Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Build native app
```
tauri build
```
